## 📖 專案簡介
WanderBill 是一個專為旅遊團體設計的簡易瀏覽器記帳工具。它能協助您與旅伴追蹤共同支出，並將複雜的債務結算過程極簡化為最少的交易次數。

## 🌟 為什麼選擇 WanderBill？

* 🚫 **無廣告體驗**：完全沒有干擾，專注於記帳。
* 💡 **操作簡單**：介面簡潔直觀，每個人都能快速上手。
* 🆓 **100% 免費**：所有功能均可免費使用，無隱藏費用或「進階版」限制。
* 🛡️ **隱私至上**：如 [數據儲存與隱私](#-數據儲存與隱私) 章節所述，您的數據儲存在本地瀏覽器中，確保絕對隱私。

## ✨ 核心功能

* 👤 **用戶管理**
  - 簡單的用戶名登入。
  - 所有數據僅存儲於您的瀏覽器中（無需註冊帳號）。
* 📂 **旅遊群組管理**
  - 可創建多個旅遊群組。
  - 輕鬆增減群組成員。
  - 行程結束後可隨時刪除群組。
* 💰 **帳單追蹤**
  - 記錄描述、幣別與金額。
  - **等額分攤**：自動在所有成員間平分帳單。
  - **自定義分攤**：可為每位成員指定具體金額。
  - 完整記錄每筆支出的日期與時間。
* 🔄 **債務簡化**
  - 自動計算「誰欠誰多少錢」。
  - 將複雜的債務網絡簡化為最少的交易次數。
  - *範例：若 A 欠 B $100，且 B 欠 C $200 → 簡化為 A 付給 C $100，B 付給 C $100。*
* 💱 **多幣別支持**
  - 支援 USD, EUR, GBP, JPY, CNY, KRW, THB, INR, AUD, CAD。
  - 自動顯示對應的貨幣符號。

## 🚀 如何使用

### 1️⃣ 初次設置
1. 在瀏覽器中開啟 [WanderBill](https://jiawenyu.github.io/WanderBill/)。
2. 輸入您偏好的 **用戶名**。
3. 點擊 **"Start Tracking"**。

### 2️⃣ 管理群組
- 進入 **Travel Groups** 標籤頁，點擊 **"Create Group"** 建立群組。
- 在群組上點擊 **"View"** 即可添加旅伴的用戶名。

### 3️⃣ 添加帳單
- 進入 **Bills** 標籤頁並選擇您的群組。
- 填寫詳細資訊（描述、金額、分攤方式）並點擊 **"Add Bill"**。

### 4️⃣ 結算債務
- 進入 **Debts** 標籤頁查看結算所有帳目所需的簡化交易清單。
- ❗ **專業建議**：如果您剛剛更新了帳單，但「簡化交易」看起來沒有變動，請 **重新整理網頁**。這能確保計算完全更新，並防止因瀏覽器顯示延遲而導致的還款錯誤。

## 💾 數據儲存與隱私

> [!IMPORTANT]
> **WanderBill 將所有數據「本地儲存」在您瀏覽器的 `localStorage` 中。**
> * 數據絕不會被發送到任何伺服器。
> * 數據在不同的瀏覽器會話（Session）之間會持續保存。
> * **若要清除數據**：請使用瀏覽器的「清除網站數據」選項或點擊登出。

## 🔄 數據同步指南

由於 WanderBill 優先採用本地儲存，請遵循以下步驟在不同裝置間同步數據：

### 如何同步
1. **指定一名「數據負責人」**：建議由一人管理主副本。
2. **匯出數據**：進入 **Data Sync** 標籤頁，點擊 **"Export Data"** 並複製 JSON 文本。
3. **分享**：透過通訊軟體（Messenger, WhatsApp 等）發送該 JSON 文本。
4. **匯入**：其他旅伴進入 **Data Sync** 標籤頁，貼上 JSON 並點擊 **"Import Data"**。

### 💡 最佳實踐
* **定期同步**：旅途中建議每天至少同步一次。
* **先行備份**：在進行重大更改前，請先匯出備份。
* **名稱一致**：確保所有裝置上的用戶名完全一致。
* **合併數據**：匯入操作會將新數據「添加」到現有數據中，而非直接覆蓋。
